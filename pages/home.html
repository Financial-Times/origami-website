Origami is a group of services, components, and tools used to help build websites. Origami aspires to create a unified style and experience for FT websites, and make web development at the FT faster.

Components

We maintain a large set of front end components which can be used to speed up development and make your project consistent with other FT websites.

  • Follow the tutorials
  • View all of our components
  • Read the component docs

Services

We provide several services for use by people at the FT, the common purpose of our services is to reduce duplicated effort across our development teams.

  • View all of our services
  • Read the service docs

Contributing

We're happy to accept contributions and suggestions across our range of components and services. We use GitHub issues and maintain a specification to help engineers contribute.

  • Propose a new component
  • Read the Origami specification
