Origami is a group of services, components, and tools used to help build websites. Origami aspires to create a unified style and experience for FT websites, and make web development at the FT faster.

Origami Team

We develop frontend tools and services for teams across the The Financial Times Group. We aim to:

  • Unify design across the FT.
  • Reduce time spent repeating work.

Get Started

To jump straight in and include Origami components in your project, see our tutorials page.

If you would like to learn about Origami components in more detail see our component documentation.

For an overview of all the tools and services we offer see our documentation page.

Contribute

We're happy to accept contributions and suggestions. We use GitHub issues and maintain a specification to help engineers contribute.
